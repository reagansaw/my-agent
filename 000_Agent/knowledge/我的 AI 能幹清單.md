# 我的 AI 能幹清單

> 這份清單跟著「外部工具整合計畫」長大。每次整合完一個工具，回來這裡加上它能做什麼。
> 目的：讓你（和未來的 AI session）一眼看到「我的 AI 現在到底會哪些具體動作」。

## 🟢 已整合的工具

### 📧 Gmail（gws CLI｜整合日期：2026-06-02）

- 掃描收件匣摘要（寄件人、主旨、日期）
- 讀取指定信件內容
- 搜尋舊信件（關鍵字、寄件人、日期）
- 建立回信草稿、轉寄信件
- 傳送新郵件
- ⚠️ 發信前會讓你確認，不會自動發出

**試試看的指令**：
- 「幫我看今天有什麼信」
- 「搜尋關鍵字『合作』的最近 10 封信」
- 「幫我回這封信：[信件 ID]，說我下週一有空」

**常用 gws 指令**：
```bash
gws gmail +triage              # 收件匣摘要
gws gmail +read --id [ID]      # 讀特定信件
gws gmail +send                # 寄信（互動式）
gws gmail +reply --id [ID]     # 回信
```

---

### 📅 Google Calendar（gws CLI｜整合日期：2026-06-02）

- 查今天 / 未來行程
- 建立新活動
- 查找空閒時段
- ⚠️ 刪除活動前會先讓你確認

**試試看的指令**：
- 「今天有什麼行程？」
- 「幫我在週五下午 3 點建一個 1 小時的會議，叫做 AI 週會」
- 「這週什麼時間有 2 小時空檔？」

**常用 gws 指令**：
```bash
gws calendar +agenda           # 未來行程總覽
gws calendar +insert           # 新增活動（互動式）
gws calendar events list --calendarId primary --timeMin [ISO時間]
```

### 📁 本機檔案（內建，無需設定）

- 讀寫任何資料夾的 Markdown、文字、程式碼檔案
- 搜尋關鍵字、整理資料夾結構
- 讀寫 Obsidian vault（直接給路徑即可）

**試試看的指令**：
- 「讀我的 Obsidian vault 裡的 [筆記名稱]，幫我整理重點」
- 「掃 200_Reference/ 資料夾，告訴我裡面有什麼素材」
