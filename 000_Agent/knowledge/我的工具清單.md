# 我的數位工具清單

> AI 在幫我寫作、推薦、評估工具時會翻這份清單。每當換工具、加新服務、決定淘汰某個東西，就更新這裡。

## 我用這些工具做這些事

### 📧 通訊與信箱

- **Gmail** — 主力信箱。選它是因為和 Google 全套生態系整合最順，搜尋功能強。

### 📅 行事曆與排程

- **Google Calendar** — 主力行事曆，和 Gmail 共用帳號，安排行程最方便。

### 📝 筆記與知識管理

- **Obsidian** — 本機 Markdown 筆記庫，Claude Code 可直接讀寫，不需要額外整合。

### 💻 程式開發與版本控制

- **GitHub** — 程式碼管理，使用官方 `gh` CLI 接 AI。

### 🎨 內容創作與 AI 工具

- **AI 工具介紹文**：使用自訂 Skill `/ai-tool-article` 起草
- 常追蹤：Seedance、AI 作圖/影片工具、創意 AI 應用

### 🌐 網頁資料抓取

- **Firecrawl** — 抓網頁內容最順的工具，走 MCP 整合（尚未完成）
- **Claude Code 內建 WebFetch** — 輕量替代方案，免費但功能較少

## 維護節奏

每個月 1 號花 5 分鐘問自己：
1. 上個月有裝新工具嗎？→ 加進清單
2. 上個月有淘汰工具嗎？→ 移到「已淘汰」
3. 有哪個工具我說不出為什麼還在用？→ 考慮砍掉
